Security and Privacy

This system must protect user data such as student details and union information.

Requiremnts:
- Users must log in securely using a username and passowrd.
- Only authorised users can access certain data.
- All personal data must be stored securely.
- The system should prevent unauthorised access.

Performance

The system must respond quickly to users and handle many requests efficiently.

Requirements:
- The system should load pages within 2 seconds.
- The system should support multiple users at the same time.
- Event search results should be displayed quickly.
- Notifications should be sent in real-time.

Reliability 

The system must work consistently without failures.

Requirements:
- The system should be available 24/7.
- The system should recover quickly from failures.
- Data should not be lost during system errors.
- Backups should be performed regularly.

Scalability

The system must handle increasing numbers of users and data.

Requirements:
- The system should support a growing number of users.
- The system should handle large amounts of event data.
- The system should maintain performance under high load.
- Cloud infrastructure should be used to scale resources.
