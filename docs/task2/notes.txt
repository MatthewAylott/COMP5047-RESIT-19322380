Security and Privacy

This system must protect user data such as student details and union information.

Requiremnts:
- Users must log in securely using a username and passowrd.
- Only authorised users can access certain data.
- All personal data must be stored securely.
- The system should prevent unauthorised access.

Performance

The system must respond quickly to users and handle many requests efficiently.

Requirements:
- The system should load pages within 2 seconds.
- The system should support multiple users at the same time.
- Event search results should be displayed quickly.
- Notifications should be sent in real-time.
